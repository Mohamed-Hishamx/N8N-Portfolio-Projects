Project Overview

This project is an automated sales monitoring and alert system built using n8n, designed to help sales and management teams track daily performance and receive instant alerts when sales fall below a defined threshold.

The workflow runs on a schedule, analyzes sales data from Google Sheets, and automatically notifies stakeholders via Telegram if performance issues are detected.
